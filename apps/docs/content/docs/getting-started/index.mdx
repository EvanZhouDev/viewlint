## Quick Start

Install and configure ViewLint

```bash tab="NPM"
npm init @viewlint/config@latest
```

```bash tab="PNPM"
pnpm create @viewlint/config@latest
```

```bash tab="Bun"
bun create @viewlint/config@latest
```

```bash tab="Yarn"
yarn create @viewlint/config
```
