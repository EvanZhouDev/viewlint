title: Manual Installation
description: Make a Custom ViewLint Setup
---

You can also manually set up ViewLint in your project.

## 1) Install packages

Install ViewLint and the built-in rules:

```bash tab="NPM"
npm install --save-dev viewlint@latest @viewlint/rules@latest
```

```bash tab="PNPM"
pnpm add --save-dev viewlint@latest @viewlint/rules@latest
```

```bash tab="Bun"
bun add --dev viewlint@latest @viewlint/rules@latest
```

```bash tab="Yarn"
yarn add --dev viewlint@latest @viewlint/rules@latest
```

## 2) Create a config file

Create `viewlint.config.ts` in your project root:

```bash
touch viewlint.config.ts
```

Add this starter config:

```ts
import { defineConfig } from "viewlint/config";
import rules from "@viewlint/rules";

export default defineConfig({
  plugins: {
    rules,
  },
  extends: ["rules/recommended"],
});
```

## 3) Run ViewLint

```bash tab="NPM"
npx viewlint https://example.com
```

```bash tab="PNPM"
pnpm dlx viewlint https://example.com
```

```bash tab="Bun"
bunx viewlint https://example.com
```

```bash tab="Yarn"
yarn dlx viewlint https://example.com
```
